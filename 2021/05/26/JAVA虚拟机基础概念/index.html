堆

new 出来的对象放在内存堆
首先是新生代的伊甸园区，然后minor gc 定期检查该（伊甸园区和survivor区）对象，检查后将有用对象将对象放入另一个 survivor 区域
伊甸园：Survivor ：Survivor = 8 : 1: 1
当Survivor 区中的年代数达到15， 则会被移入老年代，新生代放中Survivor 对象大小超过Survivor大小的50%，也会直接被放入老年代
